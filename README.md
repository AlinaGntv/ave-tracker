# Тайм-трекер

Простой одностраничный тайм-трекер на чистом HTML/CSS/JS. Данные хранятся в `localStorage` браузера.

## Как работает

- **Таймер** — введите название задачи и нажмите «Старт». Время пойдёт. Нажмите «Стоп» — запись сохранится.
- **Ручной ввод** — можно добавить задачу с указанием даты, часов и минут без использования таймера.
- **Статистика** — автоматически считает суммарное время за сегодня, вчера, неделю, месяц и год.
- **История** — показывает последние 15 записей. У каждой есть кнопка «×» для удаления.
- **Сбросить всё** — очищает всю историю.

Просто откройте `index.html` в браузере.
